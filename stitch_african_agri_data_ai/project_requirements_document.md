# African Language & Agricultural Data for AI - PRD

## Visual Identity
*   **Primary Green:** #1F7A63 (Agriculture, Trust)
*   **Light Green:** #2ECC71 (Growth)
*   **Accent Orange:** #F39C12 (Africa, Energy)
*   **Deep Black:** #0F172A (Tech)
*   **Light Gray:** #F5F5F5 (Background)
*   **Typography:** 
    *   Titles: Poppins Bold
    *   Body: Inter Regular
    *   Buttons: Semi-bold
*   **Grid:** 12 columns, max-width 1200px, 24px padding.

## Core Screens to Design
1.  **Home Page:** Hero, Trust section, 4-column Services grid, Specialization split-section, Datasets preview, and final CTA.
2.  **Services Page:** Detailed sections for Text, Audio, and Image annotation, plus the dataset creation pipeline.
3.  **Datasets Page:** Grid layout with a sidebar for filtering (Language, Type, Sector).
4.  **Languages Page:** Grid of supported languages (Dioula, Baoulé, etc.) and a section on low-resource language value.
5.  **Agriculture & IA Page:** Visual focus on crops (Cocoa, Hevea, Palm) and use cases like disease detection.
6.  **About Page:** Mission, Vision, and the "Why Africa" narrative.
7.  **Contact Page:** Project form + WhatsApp/Email contact info.

## Design Components
*   **Buttons:** Primary (Green/White), Secondary (Outline).
*   **Cards:** 12px border-radius, light shadow, 16px padding.
*   **Inputs:** Gray border, green focus state.
